Параметры коммандной строки

    1 параметр                2 параметр                        3 параметр                      4 параметр
"Путь к файлу БД" "Путь к файлу для сохранения (csv)" "разделитель полей в файле csv" "добавлять названия полей (0,1,2,3)"

--------------
Все параметры необходимо писать в кавычках "", между параметрами - пробел
Обязателен только первый параметр, остальные можно пропустить.

--------------
1ый - путь к файлу БД. 
Например "c:\db.db"

--------------
2ой - путь к файлу csv
Например "c:\db.csv"
Если он не указан, то прога автоматом создаст там же, где БД.

--------------
3ий - разделитель полей в csv
Символ - разделитель полей в файле
Если не указан - автоматом ставится ";"

--------------
4 параметр - добавлять ли заголовки (имена полей (из БД))
0 - не добавлять
1 - добавлять в файл csv первой строкой
2 - создавать файл Schema.ini "рядом" с файлом csv
3 - добавлять в файл csv первой строкой + создавать файл Schema.ini "рядом" с файлом csv
Если не указан - автоматом 2

--------------
Если необходимо задать только 1ый и 4ый, а остальные оставить по-умолчанию, то вместо значения следует вводить ""

Пример: "c:\db.db" "" "" "3"


********************************************************************************************************************************************************
Если в папке с программой нет файла Filter.ini - в файл csv выводятся все поля из БД, с названиями полей - как в БД и без сортировки
Если файл есть и в нем указано какие поля и как выводить - то файл csv будет сформирован в соответствии с этими данными

Это, если по этому полю надо сортировать данные (только одно из всех)
"Имя_поля_в_БД,Имя_поля_в_CSV,!"

Это, если без сортировки
"Имя_поля_в_БД,Имя_поля_в_CSV"

Пример:
"Col1,ИмяПользователя;!"
"Col2,ФамилияПользователя"
"Col3,ОтчествоПользователя"
В файл CSV будут выведены из БД только поля Col2, Col2 и Col3, отсортированные по полю Col1. При этом в (если в 4-ом параметре не 0), то заголовки полей
в файле csv и Schema.ini будут "ИмяПользователя,ФамилияПользователя,ОтчествоПользователя".
