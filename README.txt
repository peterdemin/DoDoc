Описание
========

DoDoc (от англ. Do - делать, Doc - документация) - инструмент для автоматизированного формирования шаблонной документации.
Руководсто пользователя содержится в файле Manual.odt и описывает базовые принципы использования DoDoc.
Описание архитектуры содержится в файле Architecture.txt.

DoDoc используется в Московском Опытно-Конструкторском Бюро "Марс" для формирования в соответствии с Единым Стандартом Программной Документации Технический Требований на алгоритмы, Спецификаций на подсистемы, Разнообразных Формуляров, Извещений и Отчетов.
DoDoc использует открытые стандарты и является свободно распространяемым кросс-платформенным проектом с открытым исходным кодом.

В общих чертах DoDoc принимает на вход файл .odt с шаблоном документа и файл .xml с данными для заполнения шаблона. На выходе - .odt файл, который можно сразу же сконвертировать в формат pdf.

Требования
==========

DoDoc полностью написан на Python, используется версия 2.6.
DoDoc работает с .odt файлами напрямую, поэтому для базового набора функциональности не требует установки дополнительного ПО.
Для ковертирования .odt в .pdf и .odg в .wmf используется StarOffice (OpenOffice, или LibreOffice). DoDoc запускает сервис StarOffice, поэтому скрипту должен быть доступен файл soffice.bin. В Linux это реализуется установкой LibreOffice из Менеджера Пакетов, в Windows необходимо добавить в переменную окружения PATH путь к папке program (Например: "C:\Program Files\OpenOffice.org 3\program;").
